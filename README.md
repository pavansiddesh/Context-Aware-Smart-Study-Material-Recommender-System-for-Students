# 📚 Smart Study Material Recommender System

## 📌 Overview

This project recommends study materials based on subject, difficulty level, and content type using content-based filtering.

## 🚀 Features

* Personalized recommendations
* Cosine similarity-based filtering
* Simple GUI using Streamlit

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app/app.py
```

## 📊 Dataset

Contains study materials categorized by subject, level, and type.
